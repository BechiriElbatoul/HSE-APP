# 🌟 GIRI - Application de Gestion Intégrée des Ressources et des Installations 🌟

## 🚀 Contexte
Bienvenue dans **GIRI** ! Ce projet a été développé lors de mon stage chez **Siemens Algérie**. L'objectif ? Créer une app mobile super pratique pour gérer les employés, les guides, les serre-fils et tout ce qui touche à la sécurité dans l'entreprise. C'est une étape importante pour moderniser les opérations de Siemens Algérie et contribuer à leur stratégie de digitalisation.

## 🎯 Objectifs
L’application a pour mission de :
- 👥 **Suivre la présence des employés**.
- 🧑‍🏫 **Organiser les guides et serre-fils**.
- 🔥 **Planifier la maintenance des extincteurs et alarmes incendie**.
- 📊 **Fournir des statistiques et rapports** via un tableau de bord centralisé.

## 🛠️ Fonctionnalités clés
1. **Gestion des employés**  
   - Suivi de la présence avec un simple bouton cliquable.

2. **Gestion des guides et serre-fils**  
   - Ajouter, modifier, supprimer, et associer à des étages spécifiques.

3. **Maintenance des équipements**  
   - Alertes automatiques pour la maintenance des extincteurs et alarmes incendie.

4. **Tableau de bord administratif**  
   - Stats en temps réel et rapports mensuels sur la présence par étage.

5. **Gestion des utilisateurs**  
   - Authentification sécurisée et gestion des rôles (admin, employé...).

## 🔒 Sécurité
- Toutes les données sensibles sont chiffrées 🔐.
- Les permissions sont gérées selon les rôles pour assurer une sécurité maximale 👮‍♂️.

## ⚡ Installation rapide
1. Clonez le repo :  
    ```bash
    git clone https://github.com/votre-utilisateur/giri.git
    ```

2. Installez les dépendances :  
    ```bash
    npm install
    ```

3. Lancez l'application :  
    ```bash
    npm start
    ```

## 💻 Technologies utilisées
- **React Native** pour l'app mobile.
- **Firebase** pour la gestion des utilisateurs et les données en temps réel.
- **Node.js** pour le back-end.
- **MongoDB** pour la base de données.
